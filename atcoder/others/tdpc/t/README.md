# TDPC

## [TDPC] T - フィボナッチ

### 問題概要

K項間フィボナッチ数列の第N項を求める。


### 方針

きたまさ法。きたまさ法は、m項間漸化式のk項目を高速に求める方法。

参考:

- [Typical DP Contest T - フィボナッチ - Algoogle](http://algoogle.hadrori.jp/tdpc/t/)
- [m項間漸化式の高速なアルゴリズム](http://nitcoder000.hatenablog.com/entry/kitamasa)

### 計算量

O( K^2 log N )


### keywords

dp, きたまさ法, 漸化式, フィボナッチ数列
