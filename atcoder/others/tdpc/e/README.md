# TDPC

## [TDPC] E - 数

### 方針

桁DP。

```
dp[k][d] = (10^k 未満の数で、十進法表記したときの各桁の数の和を D で割った余りが d であるもの)
```

として、dp のテーブルをつくる。

次に、N を上の桁から順に見ていき、N 以下での D で割り切れる個数を求める。

例えば、N = 3209 のとき、

000 -- 999 のテーブルを使って、
    - 0000 -- 0999
    - 1000 -- 1999
    - 2000 -- 2999
の範囲の個数が求められる。
00 -- 99 のテーブルを使って、
    - 3000 -- 3099
    - 3100 -- 3199
の範囲の個数が求められる。

以下同様。

### 計算量

O( log N * D )


### keywords

dp, 桁dp, mod
