# TDPC

## [TDPC] D - サイコロ

### 方針

$D = 2 ^ {d2} * 3 ^ {d3} * 5 ^ {d5}$ とする。

```
dp[n][i][j][k] := n 回目で (2^i * 3^j * 5^k) となる確率
```

として dp で計算する。ただし、 i, j, k が d2, d3, d5 より大きくなるなる場合は、min をとって d2, d3, d5 の範囲に合わせる。これによって、Dの **倍数** となる確率を計算することができる。

### 計算量

O( N * d2 * d3 * d5 )


### keywords

dp, 倍数
